# Radial_perturbation_Horndeski
Repository that contains a Mathematica Notebook as well as a text file that can be used to compute radial perturbation of a compact object in any quadratic Horndeski theory.

# Few Remarks:
In the notebook, we use the historical notation for Horndeski theory with G_2,G_4,F_4 etc and X=-1/2 \phi_\mu\phi^\mu.
